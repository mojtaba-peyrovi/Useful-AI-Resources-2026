### Feature engineering:

When we deal with a conversational LLM, each side has a certain role. The person who asks question from the LLM takes the role USER and the LLM side takes the role ASSISTANT.

```
There is one message role that is not visible when you are talking to the model in the https://chatgpt.com/ UI—the 'system' role. This role provides setup instructions that define the behavior of the model and/or guidelines on how the conversation should proceed (e.g., keep a specific tone, avoid certain topics, etc.)
```
 __HINT:__ Another name for "system prompt" is "developer message". 

 In order to give more instructions to the model, we need to go to OPENAI playground (https://platform.openai.com/playground/chat), not the traditional chatgpt. (https://chatgpt.com/)
